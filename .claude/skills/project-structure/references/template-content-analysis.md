# Content analysis

## Content types
For each content type:
- Name and purpose
- Field inventory: `fieldName` (type) - description
- Relationships to other types

## Rich text format
- Format type (Portable Text, Contentful nodes, HTML, Markdown, etc.)
- Embedded content support (images, videos, custom blocks)
- Example structure snippet

## Localization
- Locales present
- Strategy: field-level, document-level, or none
- Default locale

## Assets
- Storage location/reference pattern
- Asset metadata available
- Image transformation support

## Reusable block analysis
For each group of structurally similar blocks:
- Group name and member blocks
- Shared core fields (name, type)
- Per-block differences (extra/missing/different fields)
- Recommendation: variation candidates vs. separate slices

## Unusual patterns
- Polymorphic fields
- Deeply nested structures
- Circular references
- Any other edge cases

## Summary
- Total content types count
- Total documents count (per type)
- Key relationships diagram (simple ASCII or list)
